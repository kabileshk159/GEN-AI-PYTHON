# Rock Paper Scissors Game 

## Description
This is a simple Rock Paper Scissors game created using Python.

The user chooses one option:
- Rock 
- Paper 
- Scissors 

The computer randomly selects an option, and the program compares both choices to decide the winner.

## Features
- User input handling
- Random computer choice generation
- Win, lose, and draw conditions
- Simple Python logic

## Concepts Used
- Variables
- Conditional statements (if, elif, else)
- Random module
- User input
- Basic programming logic

## How to Run

1. Download or clone this repository
2. Open the Python file
3. Run the program
